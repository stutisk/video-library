
# galaxiaGana


## About galaxiaGana

lets you play your favourite Music

## Tech Stack and Tools

- React JS
- React Router v6
- React Context API + useReducer
- TailWind CSS
- Backend setup using [MockBee](https://mockbee.netlify.app/)
- React-toastify for Toasts
- Cloudinary for Hosting of Images
